# Grapher

## Actions

| Action                       | Keyboard Shortcut |
| ---------------------------- | ----------------- |
| Add a function               | ---               |
| Edit a function              | ---               |
| Change a function's colour   | ---               |
| Remove one or more functions | Delete            |
| Undo actions                 | Ctrl + Z          |
| Redo actions                 | Ctrl + Shift + Z  |

> On a Mac, use the modifier `Command` key instead of `Ctrl` when using shortcuts
